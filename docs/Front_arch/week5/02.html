# 初始化

初始化和项目启动方法：

# 初始化
$ mkdir egg-example && cd egg-example
$ npm init egg --type=simple
$ npm i
# 项目启动
$ npm run dev
$ open http://localhost:7001

思考：为什么 npm init egg 可以初始化 egg 项目？

npm init带参数的时候，会默认执行 create-egg 这个命令
